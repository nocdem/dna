CPUNK Whitepaper

A technical overview of the post-quantum infrastructure powering privacy-first identity, messaging, and finance — built for the quantum era.

Overview

What is CPUNK?

CPUNK is a community-driven ecosystem building privacy-preserving, quantum-resistant tools for a sovereign digital future. Every component — from identity to messaging to finance — is designed to withstand the coming quantum computing revolution while keeping users in full control of their data.

Privacy by Design

No phone numbers, no email addresses, no KYC. Your identity is cryptographic, your communications are encrypted end-to-end, and your metadata stays invisible.

Quantum Resistant

Every cryptographic operation uses NIST-standardized post-quantum algorithms. Current and future quantum computers cannot break the encryption protecting your data.

Fully Decentralized

No central servers, no corporate gatekeepers. The Nodus network distributes data across independent nodes using Kademlia routing with k=8 replication factor.

Open Source

All code is publicly auditable. The entire stack — from low-level cryptography to the desktop and mobile apps — is transparent, verifiable, and community-owned.

Cryptographic Foundation

Post-Quantum Cryptography

CPUNK uses exclusively NIST-standardized post-quantum algorithms for all cryptographic operations. Classical algorithms like RSA and ECC are vulnerable to Shor's algorithm on quantum computers. Our stack is future-proof by design.

All cryptographic primitives are implemented in pure C for maximum performance and portability. The shared crypto library is used across every component of the ecosystem.

Self-Sovereign Identity

DNA Identity

DNA (Decentralized Network Applications) includes a self-sovereign identity system. Your identity is a Dilithium5 keypair derived from a BIP-39 mnemonic seed phrase. No phone number, no email, no third party — just 24 words that you control.

Register a human-readable name on Nodus — no blockchain transaction, no central registrar, no fee. Your multi-chain wallet addresses are automatically derived from the same seed, so your entire identity — messaging, wallets, and profile — is unified under one cryptographic root. Your DNA identity is the foundation for everything in the CPUNK ecosystem.

Encrypted Messaging

DNA Connect

DNA Connect is a post-quantum encrypted messenger that operates over the Nodus network. Every message is encrypted end-to-end using Kyber1024 key encapsulation and AES-256-GCM symmetric encryption. No central server ever sees your plaintext.

Messages are stored in the DHT with a 7-day TTL, enabling asynchronous delivery without requiring both parties to be online. Group messaging uses rotating Group Encryption Keys (GEK) for forward secrecy. Zero metadata is collected — no IP logging, no contact graphs, no read receipts leaking to servers.

Network Infrastructure

DNA Nodus Network

Nodus is a pure C implementation of a Kademlia distributed hash table. It forms the backbone of the entire CPUNK ecosystem — storing identities, delivering messages, and maintaining network state without any central authority.

512-bit Keyspace

SHA3-512-based routing with Kademlia XOR distance metric. k-buckets of size 8 provide resilient peer discovery across the network.

Kademlia Routing

XOR-based distance metric for efficient peer lookup. Data is replicated to the k=8 closest nodes for high availability and fault tolerance.

Community-Operated

DNA Nodus is open source — anyone can run a node. Nodes are geographically distributed, using UDP for Kademlia peer discovery and TCP for client connections.

CBOR Wire Protocol

Compact binary serialization over custom wire frames. Two-tier protocol: Tier 1 (Kademlia DHT ops) and Tier 2 (authenticated client operations).

Native Presence

Server-side presence tracking with inter-node p_sync broadcast. Real-time online status without DHT polling overhead. 45-second TTL on remote entries.

7-Day TTL Storage

Values persist for 7 days with automatic replication to closest nodes. Up to 1 MB per value with Dilithium5 signature verification on writes.

Multi-Chain Finance

DNA Wallet

DNA Wallet is a non-custodial, multi-chain cryptocurrency wallet integrated directly into DNA Connect. Send and receive tokens across Cellframe, Ethereum, Solana, and Tron — all within your encrypted conversations.

Each chain's keys are derived from your single BIP-39 seed using standard HD key derivation paths. One recovery phrase backs up all wallets. In-chat transfers let you send crypto as easily as sending a message, with transaction details encrypted end-to-end.

Technical Architecture

Full Stack Overview

The CPUNK ecosystem is a vertically integrated stack. From post-quantum cryptographic primitives at the bottom to polished desktop and mobile apps at the top, every layer is purpose-built for privacy and quantum safety.

DNA Connect

Cross-platform (Linux, Android, Windows) user interface with dart:ffi bindings to the native C engine. Riverpod state management, Font Awesome icons, full i18n support.

DNA Engine (C)

Modular async C library with 17 domain handlers, task queue, and thread pool. Handles identity, messaging, contacts, groups, channels, wallet, and presence.

Shared Crypto

Pure C implementations of Kyber1024, Dilithium5, SHA3-512, AES-256-GCM, BIP-39, BIP-32 HD derivation, secp256k1, and Ed25519. Used by all components.
